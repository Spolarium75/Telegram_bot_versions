# Telegram Timer Bot

A Telegram bot built with Node.js that supports:

- ⏰ Manual timers
- 🔢 Transaction-based timers
- 📜 Timer history
- 🕒 Current active timer
- 💾 SQLite persistence
- 🔔 Customizable notifications

## Setup

1. Install dependencies:
   npm install

2. Create a `.env` file:
   BOT_TOKEN=your_telegram_bot_token_here

3. Run the bot:
   npm start
